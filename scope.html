Research Project Scope

Research Problem

issues with excessive smartphone use by children,
  • Children are prone to computer vision related symptoms due to prolonged smartphone usage.
  • Parents does have the ability to identify child's behavioral changes such as depression, anxiety and addiction.
  • Parents does not have a method to identify child's word usage patterns in smartphones.
  • Data extraction from smartphones has a huge impact on device performance.
Above are the Research Problems we should address.

Research Objective

Develop and implement a efficient system to smartphone usage of children.
  • Extracting data from smartphone without affecting the performance.
  • Detecting cyberbullying, sexting, and pornographic word usage patterns.
  • Detecting behavior changes like Anxiety, depression and addiction.
  • Preventing eyesight related problems caused by prolonged smartphone usage.

Methodology

The system consist of 2 main components and 4 independent components.
  • The system consist of four independent software components in two servers, one web application, and one mobile application.
  • Behavior analysis componenet and inappropriate word usage detection component is build using machine learning.
  • Face-screen distance estimation component is built using computer vision.
  • Data extraction component is developed using android and nodejs.
  • Web application dashboard is developed using Angular.
